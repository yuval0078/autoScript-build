# autoScript-build

This repository contains setup documentation and configuration files for the TouchpadExperimentManager application.

## Overview

TouchpadExperimentManager is a portable application for managing touchpad experiments. This repository provides:

- **Setup Instructions**: Complete guide for repository setup and release management
- **Configuration Files**: Example configuration files for version management and auto-updates
- **Release Process**: Step-by-step instructions for creating and publishing releases

## Quick Start

1. **Setup Guide**: See [SETUP.md](SETUP.md) for detailed instructions on:
   - Initial repository setup
   - Creating releases
   - Version management
   - Configuration file structure

2. **Configuration Files**:
   - `version.json` - Version metadata and changelog
   - `update_config.json` - Auto-update configuration

## Important Notes

- Replace `<your-user>` with your GitHub username in all URLs
- Ensure version numbers are consistent across all files
- Follow the release checklist in SETUP.md before publishing

## Files in this Repository

- **SETUP.md**: Comprehensive setup and release guide
- **version.json**: Example version metadata file
- **update_config.json**: Example auto-update configuration

## Getting Started

For first-time setup, follow the instructions in [SETUP.md](SETUP.md).

## Release Process

Creating a new release involves:
1. Building the portable application
2. Creating a git tag
3. Uploading release artifacts to GitHub
4. Updating configuration files with correct URLs

See the full process in [SETUP.md](SETUP.md).
